软件工程师基本技能

发表于 2015-09-11 更新于 2019-06-29

通用部分

语言中级

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：所有

熟练使用编程语言。熟练使用常用的类或库，熟悉IO处理、异常处理、文本时间处理、XML/JSON处理、数据库操作、网编编程等常用功能。

数据库中级

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：所有

掌握一种或多种数据库。精通SQL语言，能完成各种查询和统计功能。

类设计

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：所有

按照提供的需求，进行类设计,要求满足以下要求：

1、能够体现出面向对象编程的特点：封装、继承、重载、多态；

2、能够体现出面向对象编程的基本原则：开-闭原则、替换原则、依赖原则、接口分离原则；

3、能反应出部分设计模式的应用，但不一定要求能解释清楚设计模式定义。

基础设计模式

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：所有

了解设计模式的基本分类，能列表至少5个常用设计模式，并能阐述各自的特点和应用场景。

除单例模式外，至少在编码使用过两个别的设计模式。增强代码的可扩展、可维护性。

专题技术

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：所有

学习通用专题技术，扩展业务相关专题技术。

WEB方向基础

HTML&CSS高级

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：WEB开发

可以熟练快速的搭建各种界面结构，实现各种界面效果。

JavaScript高级

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：WEB开发

深入理解JavaScript语言特性。

JAVA WEB方向

[开始学习JAVA WEB技术]

Tomcat调优

  • 重要程度：★★★★★
  • 学习难易：★★
  • 适合人员：JAVA WEB开发

深入了解产品的体系结构、熟悉常见问题的解决办法，能够进行疑难问题解决，以及性功能调优工作

Spring/Struts/Hibernate基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：JAVA WEB开发

1、对主流框架的功能有比较全面的了解，能够使用框架进行业务系统的开发，了解常见的业务开发中的问题

2、能够熟练配置地单独或是组合使用SSH，说明基于SSH开发的配置方法和实现步骤。（说明applicationContext.xml、web.xml、hibernate.cfg.xml的配置方法）
