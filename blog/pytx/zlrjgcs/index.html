助理软件工程师基本技能

发表于 2015-09-11 更新于 2019-06-29

通用部分

编码规范

  • 重要程度：★★★★★
  • 学习难易：★★
  • 适合人员：所有

不同的语言有不同的规范，编程语言、SQL都有。熟知并遵守基本规范，如命名、注释密度、代码格式、常量定义等。

语言基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：所有

掌握开发所需语言，一般为JAVA或C/C++。不同行业和公司用的语言是不同的，这两种是范围最大的，也有用Python的。

熟悉语言的基本语法、逻辑控制，以及对象体系。

数据库基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：所有

了解数据库，掌握基本的SQL语句写法。会用一种或多种数据库管理工具。

开始学习数据库

WEB方向基础

HTML基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：WEB开发

熟悉语言的基本语法、关键字信息，能够和CSS结合，编写比较美观、复杂的网页页面。

CSS基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：WEB开发

属性(Properties)、规则(At-Rules)、伪类(Pseudo-Classes)、伪元素(Pseudo-Elements)、声明(Declarations)、单位(Units)、选择符(Selectors)等。

开始学习CSS

JavaScript基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：WEB开发

熟悉语言的基本语法、了解语言的对象体系，以及主要的功能特性。

开始学习JavaScript

JAVA WEB方向

开始学习JAVA WEB技术

Tomcat基本配制

  • 重要程度：★★★★★
  • 学习难易：★★
  • 适合人员：JAVA WEB开发

开发中常用的中间件，熟练使用Tomcat，要会多种方式部署WEB应用。

JSP/Servlet基础

  • 重要程度：★★★★★
  • 学习难易：★★★
  • 适合人员：JAVA WEB开发

熟悉Servlet/JSP的书写方法，了解决JSP和Servlet的关系，会搭建JAVA
WEB项目。
