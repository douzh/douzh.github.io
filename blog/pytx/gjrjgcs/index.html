高级软件工程师基本技能

发表于 2015-09-11 更新于 2019-06-29 分类于 知识体系

通用部分

语言高级

  • 重要程度：★★★★★
  • 学习难易：★★★★★
  • 适合人员：所有

1、熟悉语言的高级功能包的功能特性，包括安全、网络、多线程等，以及扩展的语言包。

2、能够运用高级功能包进行业务系统的开发、测试工作。

java高级

1、熟悉目前比较主流的Java开源模块，如日志、XML处理、池管理器等，并能运用到业务系统的开发中，同时能够进行相关问题的解决。

2、掌握JVM相关知识，了解Java语言的设计原理，具有基本的调优排查问题的能力。

3、通过SCJP认证

数据库高级

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：所有

学习最好的方式就是参加大量大型项目的设计工作。

索引应用：

  • 索引何时生效
  • 索引类型
  • 如何强制使用索引
  • 如何判断是否使用索引
  • 使用索引时的注意事项

数据库设计：

  • 考虑目前的生产环境；
  • 考虑旧数据兼容性；
  • 定义标准的对象命名规范；
  • 遵守3范式；
  • 高级冗余，适当的引入冗余设计，提高性能、降低业务实现难度；
  • 避免使用触发器，文档化管理；
  • 主键的生成方式；
  • 物理存储、逻辑存储规划；
  • 基本索引；
  • 数据库服务基本参数设定；
  • 实体关系设计

数据库调优：

索引、分割数据、产品参数调整、避免热数据、缓存

高级设计模式

  • 重要程度：★★★★★
  • 学习难易：★★★★★
  • 适合人员：所有

掌据至少15个设计模式，能阐述各自的特点和应用场景。

在系统架构设计中，能够应用设计模式。

最好的学习方式就是看开源项目的实现方式。

专题技术

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：所有

学习通用专题技术，扩展业务相关专题技术。

WEB方向

集群技术

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：所有

了解集群的原理，掌握中间件集群时需要注意的问题（负载均衡、Session同步等），至少掌握一个中间件的集群配置达到熟练程度。

JAVA WEB方向

Spring/Struts/Hibernate高级

  • 重要程度：★★★★★
  • 学习难易：★★★★
  • 适合人员：JAVA WEB开发

1、对主流框架的功能、体系结构有比较深入的了解；能够对框架使用中的疑难问题进行分析、解决。

2、对SSH有较深入的研究，了解SSH的架构设计，了解SSH的一些特性的实现原理(如Spring的AOP、IoC,Struts的MVC，Hibernate中的Dialect等）

3、在系统设计时能够借鉴SSH的设计思想。
