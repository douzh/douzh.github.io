如何学好C语言

发表于 2015-01-12 更新于 2019-06-23 分类于 编程语言 ， C语言

为什么要学C语言

首先，要明白为什么要学C语言，C语言可以用来做什么。

每种语言都有其用途和领域，C语言适用于如下领域：

  1. 操作系统开发。
  2. 嵌入式开发。
  3. 驱动开发。
  4. 音视频开发。
  5. 性能苛刻的程序开发。

相对于其它语言，C语言的最大优势是可以操作计算机低层的功能。

另外C语言因为有指针和函数指针的存在，可以灵活构建程序结构，确实是一门好语言。

如何学好C语言

学好C语言不容易，不是C语言不容易学，而是学习C语言要学习很多计算机基础知识。

计算机编程相关基础知识：

  1. 学习计算机体系结构：至少明白内存、CPU、中断等基本计算机原理，明白程序是怎么用0和1执行的。
  2. 学习汇编语言：明白计算机的基础指令和0、1数据的关系，明白编程的本质是怎么回事。
  3. 学习编译原理：明白高级语言的实现原理，比如栈和堆的本质实现，高级语言编译成汇编语言的逻辑等。
  4. 学习操作系统知识：了解操作系统实现的各方面，如进程、线程、文件、内存管理等。

除了学计算机基础知识，还要学编程的相关知识，学任何语言都要学：

  1. 数据结构：非常基础的编程知识。
  2. 设计模式：想做软件开发，要好好学习的基础知识。
  3. 软件架构：想做IT行业人牛人，相当高工、架构师要学习的知识。

学习C语言的步骤：

  1. 基本语法：任何语言的基本语言都差不多，很好学。
  2. 指针、函数指针：如果没有很好的计算机基础，到这就卡住了。
  3. 结构体与共用体：配合指针，基本面向对象的思想都能实现。
  4. 专题开发学习：文件操作、多进程开发、驱动开发、操作系统开发等，实际做点东西。

学习资料推荐

《算法导论》

学习数据结构和算法的好书。

《30天自制操作系统》

《自己动手写操作系统》

用实例讲解计算机结构、操作系统、汇编、C语言的相关知识，最好自己写一个操作系统，实践一下书中内容。

《CC深层探索》

深度研究C语言的实现，有助于理解C语言原理，精通C语言。
