Java中的Enum用法介绍

  1. 关于Java Enum:

学过C/C++等语言的人，应该都对Enum类型略知一二。Enum一般用来表示一组相同类型的常量。如性别、日期、月份、颜色等。对这些属性用常量的好处是显而易见的，不仅可以保证单例，且比较时候可以用”==”来替换equals。是一种好的习惯。JDK1.5之前没有Enum这个类型，那时候一般用接口常量来替代。有了JavaEnum之后，可以更贴近的表示这种常量。
