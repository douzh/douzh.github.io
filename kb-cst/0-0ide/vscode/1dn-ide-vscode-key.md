# VS Code 快捷键

## 基础

| shortkey      | 说明         | 命令 |
| ------------- | ------------ | ---- |
| Ctrl+Shift+P  | 打开命令面板 |      |
| Ctrl+Shift+E  | 资源管理     |      |
| Ctrl+Shift+F  | 搜索         |      |
| Ctrl+Shift+X  | 扩展         |      |
| Ctrl+Alt+箭头 | 列模式       |      |
